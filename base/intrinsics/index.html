    wasm_memory_atomic_wait32 :: proc(ptr: ^u32, expected: u32, timeout_ns: i64) -> u32 {…}
     

    timeout_ns is maximum number of nanoseconds the calling thread will be blocked for. A negative value will be blocked forever. Return value: 0 - Indicates that the thread blocked and then was woken up. 1 - The loaded value from ptr did not match expected, the thread did not block. 2 - The thread blocked.
